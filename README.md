# telecom-churn-intelligence-system
# 📉 Customer Churn Prediction & Retention Intelligence System

An end-to-end Machine Learning and Business Intelligence project focused on predicting telecom customer churn and identifying high-risk customers for retention strategies.

---

# 🚀 Project Overview

This project combines:

- SQL Analysis
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning
- Explainable AI (SHAP)
- Power BI Dashboarding
- Streamlit Deployment

to create a complete customer retention intelligence system.

---

# 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SHAP
- Power BI
- Streamlit
- MySQL

---

# 📊 Key Features

✅ Customer churn prediction using Machine Learning  
✅ SHAP explainability for churn drivers  
✅ High-risk customer identification  
✅ Interactive Power BI dashboard  
✅ Streamlit web application  
✅ End-to-end ML workflow

---

# 🤖 Machine Learning Models

| Model | ROC-AUC |
|---|---|
| Logistic Regression | 0.8373 |
| Random Forest | 0.8168 |
| Tuned XGBoost | 0.8218 |

---

# 🔥 Key Insights

- Customers with low tenure showed higher churn probability
- Fiber optic users had significantly higher churn risk
- Long-term contracts reduced churn substantially
- Electronic check payment users showed elevated churn behavior
